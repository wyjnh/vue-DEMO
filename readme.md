# 基于Vue的日常工作

## server 功能介绍
* koa路由模块化
* 数据库增删改查
* 访问接口 指向 html 页面 （服务端渲染）
* koa-body 文件上传到指定文件夹 下载服务端文件
* koa-session
* 登录登出session验证 redirect重定向

## view 功能介绍
* vuecli
* markdown 插件导入
* vuex
